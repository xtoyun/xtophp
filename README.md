#### 介绍
XTOPHP是基于ThinkPHP5.1的后台开源框架
之前版本经过重新整理，采用layui前端，快速开发。

XTOPHP快速开发框架V1.0.20200509

1、后台界面调整统一，便于二次开发。

2、增加组件接口，更易于扩展功能。

3、统一api调用数据，适应对接公众号、小程序等入口

4、增加应用商店，便于更快速购买应用。
